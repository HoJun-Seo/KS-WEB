# 과제 #3
-----
## 준비물
- JDK(>=8)
- gradle or IntelliJ
## 과제 목표
- 학생 수강 신청 웹 페이지를 완성해보자.
  - 학생, 강의, 수강 신청 내역등의 정보 수정, 삭제 기능
  - 학생이 수강 신청을 완료 했을 경우 현재 수강 가능 인원 변동
  - 수강 신청 목록 페이지를 통해 내역을 삭제 했을 경우 해당 강의 수강 가능 인원 변동
  - 수강 신청이 완료된 강의나 학생 정보 자체를 삭제 하려고 할 경우 해당 요청 거부
